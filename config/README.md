# Gateway runtime config

Domain-specific settings live in the repo root:

- **`config.yaml.example`** — committed template (placeholders)
- **`config.yaml`** — generated by `./scripts/generate-env.sh` (gitignored)

Compose mounts `./config.yaml` → `/config/config.yaml` in the gateway container.

Go package defaults for production hostnames (`admin_defaults.go`) apply only when YAML leaves `admin.hosts` empty.
